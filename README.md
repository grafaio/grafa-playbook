# The Grafa Developer's Playbook

This repo contains the Grafa Developer's Playbook, a detail of how we design, build, and maintain software at Grafa. It is an evolving guidebook that is intended to support, inform, and educate through team-wide standards (both enforced and accepted), best practices, tutorials, and reference.
